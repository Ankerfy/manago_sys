`easing`（缓动函数）是动画中控制**速度变化曲线**的核心概念。它决定了动画是从头到尾匀速，还是先快后慢、先慢后快等。

在 CSS 和 JavaScript 动画（如 NProgress）中，`easing` 函数通过改变“时间 vs. 进度”的关系来影响视觉感受。

---

## ✅ 常见 `easing` 函数名及其含义

以下是 CSS 和动画系统中最常见的缓动函数名称及其行为解释：

| 函数名            | 含义             | 动画行为描述             | 适用场景                   |
| ----------------- | ---------------- | ------------------------ | -------------------------- |
| **`linear`**      | 线性             | 速度恒定，从头到尾一样快 | 极少使用，显得机械、不自然 |
| **`ease`**        | 缓入缓出（默认） | 先慢 → 中间快 → 结尾慢   | 最常用，视觉最自然         |
| **`ease-in`**     | 缓入             | 开始慢，结束快           | 适合“出现”类动画           |
| **`ease-out`**    | 缓出             | 开始快，结束慢           | 适合“消失”或“收起”动画     |
| **`ease-in-out`** | 缓入 + 缓出      | 两端慢，中间快           | 比 `ease` 更对称平滑       |

---

## ✅ 详细解释每个函数

### 1. `linear`

- **公式**：`y = x`
- 速度完全均匀，没有加减速。
- ❌ 缺点：看起来“生硬”、“机器人感”，不符合现实物体运动规律。
- ✅ 仅用于特殊需求，如循环旋转图标。

```css
transition: all 0.3s linear;
```

---

### 2. `ease`

- **等效贝塞尔曲线**：`cubic-bezier(0.25, 0.1, 0.25, 1.0)`
- 开始稍慢，中间加速，结尾减速。
- ✅ 默认值，**最自然的动画曲线**，推荐大多数情况使用。

```css
transition: all 0.3s ease; /* 默认值，可省略 */
```

---

### 3. `ease-in`

- **等效贝塞尔曲线**：`cubic-bezier(0.42, 0, 1.0, 1.0)`
- 动画开始时非常缓慢，结束时突然加速。
- ✅ 适合“物体从远处飞入”或“淡入”效果。
- ❌ 不适合进度条（用户会以为卡住了）。

```css
transition: opacity 0.3s ease-in;
```

---

### 4. `ease-out`

- **等效贝塞尔曲线**：`cubic-bezier(0, 0, 0.58, 1)`
- 开始很快，然后逐渐减速到停止。
- ✅ 最符合“惯性”物理直觉，适合菜单收起、提示框消失等。
- ✅ 也常用于按钮点击反馈。

```css
transition: transform 0.2s ease-out;
```

---

### 5. `ease-in-out`

- **等效贝塞尔曲线**：`cubic-bezier(0.42, 0, 0.58, 1)`
- 对称曲线：开始慢 → 中间快 → 结尾慢。
- ✅ 比 `ease` 更平滑对称，适合来回切换的动画。
- ✅ 可用于进度条，但略显“拖沓”。

```css
transition: width 0.4s ease-in-out;
```

---

## ✅ 高级：`cubic-bezier(x1, y1, x2, y2)` 自定义曲线

你可以使用贝塞尔曲线创建自己的 `easing` 效果：

| 示例                                    | 含义                 | 效果                     |
| --------------------------------------- | -------------------- | ------------------------ |
| `cubic-bezier(0.1, 0.9, 0.2, 1.0)`      | 弹性缓出             | 类似弹簧，结尾轻微回弹   |
| `cubic-bezier(0.68, -0.55, 0.27, 1.55)` | 强弹性               | “弹跳”效果，适合趣味动画 |
| `cubic-bezier(0.4, 0, 0.2, 1)`          | 标准 Material Design | Google 推荐的流畅动画    |

> 💡 工具推荐：[https://cubic-bezier.com](https://cubic-bezier.com) 可视化调试贝塞尔曲线

---

## ✅ 在 NProgress 中的应用建议

```js
NProgress.configure({
  easing: 'ease', // ✅ 推荐：自然流畅
  // easing: 'ease-out', // ✅ 也可用，收尾更柔和
  // easing: 'cubic-bezier(0.4, 0, 0.2, 1)' // Material 风格
})
```

- ❌ 避免使用 `linear`（太机械）
- ❌ 避免使用 `ease-in`（开头太慢，用户以为卡住）
- ✅ 推荐使用 `ease` 或 `ease-out`

---

## ✅ 总结：如何选择？

| 目标             | 推荐 `easing`           |
| ---------------- | ----------------------- |
| 通用动画（推荐） | `ease`                  |
| 平滑收尾         | `ease-out`              |
| 对称往返动画     | `ease-in-out`           |
| 快速响应感       | `ease-out` + 短 `speed` |
| 趣味/品牌动画    | 自定义 `cubic-bezier`   |

---

✅ **一句话口诀**：

> “**慢进快出看 `ease-in`，快进慢出用 `ease-out`，自然流畅选 `ease`，对称平滑上 `ease-in-out`。**”

掌握这些 `easing` 函数，你就能让进度条、按钮、页面切换等动画更具“生命力”和专业感。
